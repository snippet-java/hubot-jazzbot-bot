# hubot-shipit

A hubot script

## Installation

In hubot project repo, run:

`npm install hubot-jazzbot-bot --save`

Then add **hubot-jazzbot-bot** to your `external-scripts.json`:

```json
[
  "hubot-jazzbot-bot"
]
```

## Sample Interaction

```
user1>> hubot bot add cf https://cfclient.mybluemix.net
hubot>> bot cf added
```
